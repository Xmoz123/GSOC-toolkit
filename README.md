**Asset Canvas Toolkit**

A web-based toolkit that allows users to drag and drop assets onto a whiteboard canvas. This project provides a streamlined interface for visual organization and spatial data management.

**Features**
1. Drag & Drop Interface: Easily add and move assets across a responsive whiteboard canvas.
2. Properties Pane: Real-time controls to scale assets, add text bubbles, and adjust positioning.
3. Data Export: Download the precise coordinate data and properties of all assets as a structured JSON file.
4. Manual Precision: Options for manual input of coordinates for pixel-perfect placement.

**Tech Stack**
1. Framework: Angularv21
2. Language: TypeScript / HTML5 / CSS3
3. Data Format: JSON

**Project Structure**
1. `src/app`: Contains the core logic for the canvas and properties pane.
2. `src/assets/`: Default icons and images used within the toolkit.
3. `styles.css`: Custom styling for the whiteboard environment.

**How to Use**
1.  Select an asset from the library.
2.  Drag it onto the canvas.
3.  Use the **Properties Pane** on the right to resize or add labels.
4.  Click **Export JSON** to save the layout configuration.
